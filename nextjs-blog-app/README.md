# Next.js 博客系统

这是一个基于 Next.js 的博客系统示例，用于展示文章列表、文章详情、分类、标签和评论等常见博客功能。

## 主要功能

- 博客文章首页
- 文章详情页
- Markdown 内容渲染
- 分类页面
- 标签页面
- 本地模拟评论
- 关于页面

## 技术栈

- Next.js
- React
- Tailwind CSS
- React Markdown
- remark-gfm

## 本地运行

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

然后在浏览器打开：

```text
http://localhost:3000
```

## 说明

当前评论和文章数据使用本地模拟数据，适合作为 Next.js 页面路由、Markdown 渲染和博客结构练习示例。
