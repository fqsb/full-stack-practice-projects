# 校园活动管理系统前端

这是校园活动管理系统的前端部分，使用 React 和 Ant Design 构建。

## 主要功能

- 登录与注册
- 首页与关于页
- 活动列表展示
- 活动搜索、状态筛选、时间筛选
- 活动详情查看
- 活动报名
- 报名记录列表
- 活动审批列表
- 活动分类管理
- 用户列表管理
- 个人资料查看与编辑

## 技术栈

- React
- React Router
- Ant Design
- Axios
- Moment.js

## 本地运行

安装依赖：

```bash
npm install
```

复制环境变量示例：

```bash
copy .env.example .env
```

启动开发服务器：

```bash
npm start
```

默认后端接口地址配置在 `.env.example` 中：

```env
REACT_APP_API_BASE_URL=http://localhost:5000/api
```
